Things I've built/researched: 
### [Enagrams](https://enagrams.com) — Memory Infrastructure for AI Agents

> *Plaid, but for AI cognition.*

Building the portable memory and context layer for AI applications. Enagrams solves the core limitation of today's agents: **statelessness**.

```
┌─────────────────────────────────────────────────────────┐

│   Episodic       Semantic        Vector + Graph         │

│       │              │                │                 │

│                      │                                  │
│            Cross-Session Context                        │
│              Propagation Layer                          │

│              ▼               ▼                          │
│          Agents          Copilots                       │

```

Designing infrastructure for **persistent, queryable agent memory** — knowledge graph construction, episodic/semantic memory retrieval, and cross-session context propagation for autonomous systems. API-first, embeddable memory primitives for developers building LLM-native applications that need to **remember**.
